```bash

# MySQL Login
$ mysql -u <USERNAME> -p

$ ssh -l <USERNAME> afs8.njit.edu
```

### Dependencies 

MySQLConnector for Python

### MySQL Database Setup

create an "config.ini" file which includes:

[mysql]
host = HOSTNAME
database = DATABASENAME
user = USERNAME
password = PASSWORD
